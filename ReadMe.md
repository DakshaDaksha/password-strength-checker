# Password Strength Checker 

## Description
This is a Pyhton-based Password Strength Checker that evaluates the security of a password based on multiple criteria:
- Length of the password (minimum 8 characters).
- Use of uppercase and lowercase letters.
- Inclusion of numbers.
- Inclusion of special characters (@, #, $, %, ^, &, +, =, *, !).

The program classifies password as:
- **Weak**
- **Medium**
- **Strong**

---

## Features 
- Easy-to-use console program.
- Provides instant feedback on password strength.
- Helps understand basic password security concepts.
